WordPress Plugin to show profile edit backend on post or page with shortcode

== Description ==

Show profile edit backend on post or page through shortcode, without coding or hacks. Great for membership site.


== Installation ==

This section describes how to install the plugin and get it working.

See <a href="http://codex.wordpress.org/Managing_Plugins#Installing_Plugins">WordPress Codex Plugins Documentation</a>

insert [LOGIN] or [EDITPROFILE] to your post or page. 

You ready to go!
